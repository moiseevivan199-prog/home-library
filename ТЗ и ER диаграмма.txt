# ТЕХНИЧЕСКОЕ ЗАДАНИЕ

## Система учёта книг в домашней библиотеке

### 1. Цель разработки
Создать приложение для хранения списка книг с возможностью добавлять, удалять и отмечать прочитанные.

### 2. Функциональные требования

| № | Функция | Описание |
|---|---------|----------|
| 1 | Добавить книгу | Ввести название и автора |
| 2 | Удалить книгу | Убрать книгу из списка |
| 3 | Отметить прочитанное | Поставить галочку "прочитано" |
| 4 | Показать все книги | Вывести весь список |

### 3. хранится в таблице

| Поле | Тип | Ограничение |
|------|-----|-------------|
| ID | число | Автоматически |
| Название | текст | Не пустое |
| Автор | текст | Не пустое |
| Прочитано | да/нет | По умолчанию "нет" |

### 4. ER-диаграмма

```
┌─────────────────┐
│      КНИГА      │
├─────────────────┤
│  ID             │
│  Название       │
│  Автор          │
│  Прочитано      │
└─────────────────┘
```

### 5. Сохранение данных
Все изменения автоматически сохраняются в браузере.
