📚 Домашняя библиотека

Сайт для учёта книг в домашней библиотеке.

 О проекте:

Веб-приложение, которое помогает хранить список книг, отслеживать прочитанное и управлять домашней коллекцией.

 Возможности:

- Добавление и удаление книг
- Отметка о прочтении
- Просмотр всего списка

 Запуск:

Откройте `index.html` в браузере или разверните файлы на любом веб-сервере.

 Технологии:

Проект выполнен с использованием языка програмирования html.
